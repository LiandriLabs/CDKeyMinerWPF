# CD Key Miner WPF

This is the original CD Key Miner client for Windows, written in .NET4 and WPF.

## Installation

Download the latest setup file from the menu in your mining dashboard on [https://app.cdkeyminer.com/](https://app.cdkeyminer.com/)
